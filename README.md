# Number Guessing Game

A simple Python command-line game where the player guesses a random number with helpful hints after each attempt.

## Features

- Random number generation
- Input validation
- Hint after every wrong guess
- Attempt counter
- Replay option

## Concepts Practiced

- Loops
- Conditional statements
- Functions
- Random module
- User input validation

## Run Locally

```bash
python number_guessing_game.py
```

## Future Improvements

- Add difficulty levels
- Save best score
- Add a simple graphical interface
